# AI 产品经理且曼学院 · 知识库

> 且曼前瞻学院 AI 产品经理课程的交互式知识库网页

## 📖 简介

这是一个基于课程笔记构建的 **交互式知识可视化网页**，将 AI 产品经理课程内容整理为可视化、可交互的知识体系。

### 主要功能

- **力导向知识图谱** — PM 工作流程节点的交互式关系图，支持拖拽、缩放、点击跳转
- **课程笔记 Hub** — 按天/周组织的笔记系统，支持总览、详情、释义多种视图
- **Apple 风格沉浸式阅读** — 以天为单位的全页面阅读体验，交替明暗场景
- **内联知识可视化** — 自动识别知识点中的流程（→）、象限、对比等模式，生成 SVG 图表
- **PM 流程详解** — 10 个核心节点的完整知识体系（需求→设计→开发→测试→上线→运营）
- **暗色/亮色主题** — 自适应系统主题，支持手动切换

## 🚀 使用方式

直接在浏览器中打开 `index.html` 即可，无需安装任何依赖。

```bash
# 本地预览
open index.html

# 或使用本地服务器
python3 -m http.server 8080
```

## 🛠 技术栈

- **纯前端** — 单文件 HTML，零依赖
- **Canvas 2D** — 力导向图渲染与物理模拟
- **TailwindCSS CDN** — 样式框架
- **Vanilla JS** — 所有交互逻辑
